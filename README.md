# Portfolio (выпускной проект №1)
 Автор: Егоричев Александр Дмитриевич;


Stack:
 - Gulp 4.0
 
Getting started:

1. clone this repo
2. cd path/to/
3. npm install gulpjs/gulp-cli -g  // Install the latest Gulp CLI tools globally
4. npm install
6. run "gulp" command to start
7. gulp "build"
8. npm start command to start server
